# **App Name**: Vicious Wizard

## Core Features:

- Dynamic Character Summoning: Trigger and display animated vicious wizard or dark fairy overlays dynamically on the screen.
- Arcane Event Log: Record and display a real-time, time-stamped log of magical triggers and user interactions.
- Grimoire Persistence: Persist application data, including historical event logs, using the browser's local storage for a consistent session.
- Vicious Trait Generator Tool: An AI-powered tool that uses reasoning to generate aggressive traits or dark backstories for summoned characters.
- Sorcerer Control Hub: A responsive interface with dedicated buttons for summoning entities and managing the magic overlay settings.

## Style Guidelines:

- Primary background: A deep, dark obsidian (#080b14) to emphasize the luminous magical elements.
- Primary accent: A vibrant neon violet (#7c3aed) for buttons and active UI components.
- Secondary accent: Aggressive pink (#ec4899) and gold (#fbbf24) for gradient headers and character highlights.
- Headlines: 'Syne' font for a bold, aggressive, and modern mystical appearance.
- Log text: 'DM Mono' for precise, technical event reporting and data labels.
- Character representation: Use high-contrast emojis (🧙, 🧚) paired with glowing shadow effects.
- Organization: Card-based modules using translucent overlays (rgba(255,255,255,0.03)) with thin borders.
- Overlay transitions: Smooth cubic-bezier slide-in/slide-out animations for character entries.
- Visual effects: Subtle 'bob' and 'shimmer' animations to breathe life into the magical elements.
